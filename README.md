# вҡҷпёҸ Installation

Clone the repository

```bash
git clone https://github.com/9397karan/React_Js_Blogg_App.git
```

Go inside the project

```bash
cd React_Js_Blogg_App
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
